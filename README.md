## INTRO
윈도우 유저들을 위해 제작한 **(구)** 교내 학교생활 관리 서비스 Xquare의 **보조 클라이언트**입니다.  
  
Chrome 확장 프로그램 기반으로 제작하였으며, 급식 확인이나 주말 잔류 설정 등 간단한 작업이 가능합니다.  
  
제가 프론트엔드 엔지니어로써 성장하는 데 기반을 다져 준 프로젝트이며, 서비스 기간동안 약 **20+** 명의 사용자가 접속하였습니다.
