# Xquare in Chrome Extension
항상 모바일로 급식 확인, 벌점 확인 등을 하는게 불편해서 만든 스퀘어 확장프로그램입니다.  
내 정보, 급식 정보, 시간표를 간단히 확인할 수 있으며, 신청 기능과 기타 스퀘어 기능들을 추가해가고 있습니다.  

불편한 점이 있으시다면 dbrrlwns1127@dsm.hs.kr로 문의해주세요.
